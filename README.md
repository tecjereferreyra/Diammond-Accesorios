# Diammond-Accesorios
Página Web pensada específicamente para la visualización y consulta del catálogo de productos de mi cliente. El proyecto está conformado en Visual Community, con un entorno .net 8. Incluye un frontend con HTML,CSS Y JAVA SCRIPT. Y un backend, que maneja una conexión entre una BDD en SQLSERVER y una API en WEBCOREAPI. Utilizando el lenguaje C#
